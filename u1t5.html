Esquema de traducción

Un esquema de traducción es una gramática independiente del contexto en la que se asocian atributos con los símbolos gramaticales y se insertan acciones semánticas encerradas entre llaves { } dentro de los lados derechos de las producciones. Los fragmentos de código insertados se denominan acciones semánticas. Estos fragmentos actúan, calculan y modifican los atributos asociados con los nodos del árbol sintáctico.

Para poder aplicar un esquema de traducción, se debe construir el árbol sintáctico y después aplicar las acciones empotradas en las reglas en el orden de recorrido primero-profundo. Si la gramática es ambigua, una frase podría tener dos árboles y la ejecución de las acciones para ellos podría dar lugar a diferentes resultados.

Un ejemplo de un esquema de traducción:

L -> E n { print(E.val); }
E -> E1 + T { E.val = E1.val + T.val; }
E -> T { E.val = T.val; }
T -> T1 * F { T.val = T1.val * F.val; }
T -> F { T.val = F.val; }
F -> ( E ) { F.val = E.val; }
F -> digit { F.val = digit.lexval; }

En este esquema de traducción, cada producción de la gramática tiene asociada una acción semántica que se ejecuta cuando se aplica esa producción. Por ejemplo, en la producción E -> E1 + T { E.val = E1.val + T.val; }, la acción semántica { E.val = E1.val + T.val; } se ejecuta cuando se aplica la producción, y calcula el valor de E como la suma de los valores de E1 y T4.

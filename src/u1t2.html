1.2 Acciones semánticas de un analizador sintáctico

Las acciones semánticas de un analizador sintáctico son operaciones o instrucciones que se ejecutan durante el análisis de la sintaxis de un programa. A diferencia de las acciones sintácticas, que se centran en la estructura gramatical del código, las acciones semánticas se enfocan en el significado y el comportamiento del programa.

Las acciones semánticas pueden agruparse en:


  • Sentencias de Declaración: Completar la sección de tipos de la Tabla de Símbolos.
  • Sentencias “ejecutables”: Realizar comprobaciones de tipos entre los operandos implicados.
  • Funciones y procedimientos: Verificar que los parámetros reales de una función sean coherentes con los parámetros formales.
  • Identificación de variables: Verificar que las variables utilizadas estén declaradas y sean del tipo correcto.

Las acciones semánticas de un analizador sintáctico también pueden incluir:


  • Acceso a la tabla de símbolos: Para hacer parte del trabajo del analizador semántico.
  • Chequeo de tipos: Del analizador semántico.
  • Generación de código intermedio.
  • Generación de errores: Cuando se producen.
  • Optimización.
  • Generación de código de máquina.
